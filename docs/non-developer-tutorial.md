# 카카오워크 대화 백업 튜토리얼

이 문서는 GitHub나 PowerShell을 잘 모르는 사용자를 위한 안내입니다.
터미널 명령어를 입력하지 않아도 됩니다.

## 준비물

- Windows 10 또는 Windows 11 PC
- Windows용 카카오워크 PC 앱
- 백업할 대화방을 볼 수 있는 카카오워크 계정

## 1. 프로그램 다운로드

1. 아래 링크를 클릭합니다.
   [kakaowork-backup ZIP 다운로드](https://github.com/kangjung/kakaowork-backup/archive/refs/heads/main.zip)
2. 브라우저가 다운로드를 막거나 확인 창을 띄우면, 이 저장소를 신뢰할 수 있을 때만 다운로드를 계속합니다.
3. 보통 `다운로드` 폴더에 `kakaowork-backup-main.zip` 파일이 저장됩니다.

GitHub 화면에서 직접 받을 때는 파란색 **Code** 버튼을 누른 뒤 **Download ZIP**을 선택하면 됩니다.

## 2. 압축 풀기

1. `kakaowork-backup-main.zip` 파일을 마우스 오른쪽 버튼으로 클릭합니다.
2. **압축 풀기** 또는 **모두 추출**을 선택합니다.
3. 압축을 푼 폴더를 엽니다.

압축 파일 안에서 바로 실행하지 말고, 반드시 압축을 푼 뒤 실행하세요.

## 3. 카카오워크 준비

1. 카카오워크 PC 앱을 실행합니다.
2. 잠금 화면이 보이면 PIN을 입력해 잠금을 해제합니다.
3. 백업할 대화방을 클릭해서 메시지가 보이는 상태로 둡니다.
4. 카카오워크 창을 최소화하지 않습니다.

## 4. 백업 프로그램 실행

1. 압축을 푼 폴더에서 `Run-Exporter.cmd`를 더블 클릭합니다.
2. Windows 보안 경고가 나오면 내용을 확인한 뒤, 신뢰할 수 있을 때만 실행합니다.
3. 프로그램 창이 열리면 대화방 이름과 저장 폴더를 확인합니다.
4. **최대 스크롤 횟수**를 확인합니다. 처음에는 기본값 `500`을 그대로 두는 것을 권장합니다.
5. **반출 시작** 버튼을 누릅니다.

![카카오워크 대화 반출 실행 화면](images/exporter-ui.png)

## 5. 최대 스크롤 횟수 고르기

**최대 스크롤 횟수**는 프로그램이 카카오워크 대화방을 위로 몇 번까지 넘겨 볼지 정하는 **상한선**입니다.
대화방의 처음까지 도달하면 이 숫자를 다 채우지 않아도 자동으로 멈춥니다.
값을 크게 잡아도 끝까지 가면 알아서 종료되지만, 대화량이 아주 많은 방에서는 실행 시간이 길어질 수 있습니다.

| 대화방 상황 | 추천 값 |
| --- | --- |
| 최근 대화만 필요함 | `100` ~ `300` |
| 일반적인 업무 대화방 | `500` |
| 몇 달 이상 쌓인 대화방 | `1000` 이상 |
| 아주 오래된 대화까지 필요함 | `2000` 이상 |

잘 모르겠거나 전체 대화를 최대한 저장하고 싶으면 숫자를 넉넉하게 잡아도 됩니다.
다만 너무 오래 걸리는 것을 피하고 싶을 때 위 표를 참고하세요. 결과가 너무 짧으면
숫자를 더 크게 해서 다시 실행하면 됩니다. 너무 오래 걸리면 **중지** 버튼을 눌러
지금까지 수집한 내용만 저장할 수 있습니다.

## 6. 실행 중 주의사항

- 백업 중에는 카카오워크 창을 조작하지 마세요.
- 다른 대화방으로 이동하지 마세요.
- 카카오워크 창을 최소화하지 마세요.
- 대화량이 많으면 몇 분 이상 걸릴 수 있습니다.
- 중간에 멈추려면 프로그램 창의 **중지** 버튼을 누릅니다.

## 7. 결과 확인

지정한 저장 폴더에서 아래 파일을 확인합니다.

| 파일 | 설명 |
| --- | --- |
| `대화방이름_날짜시간.html` | 브라우저로 열어 읽는 백업 문서 |
| `대화방이름_날짜시간.json` | 검색이나 후처리에 쓸 수 있는 데이터 |
| `대화방이름_날짜시간_images` 폴더 | 화면에서 캡처한 이미지 |

일반적으로는 `.html` 파일을 더블 클릭해서 확인하면 됩니다.

## 8. 여러 대화방 한 번에 보기

대화방을 여러 개 백업했다면 `Open-BackupViewer.cmd`를 더블 클릭해 통합 뷰어를 열 수 있습니다.

![카카오워크 통합 백업 뷰어](images/backup-viewer-ui.png)

1. `Open-BackupViewer.cmd`를 더블 클릭합니다.
2. 브라우저가 열리면 왼쪽 채팅방 목록 아래의 **백업 폴더 선택**을 누릅니다.
3. 백업 결과가 들어 있는 `exports` 폴더나 여러 백업 폴더를 모아 둔 상위 폴더를 선택합니다.
4. 왼쪽 채팅방 목록에서 대화방을 선택합니다.
5. 상단에서 **현재 방** 또는 **전체** 검색 범위를 선택합니다.
6. 넓은 검색창에 대화방 이름, 사람 이름, 메시지 내용을 입력한 뒤 **검색**을 누르거나 Enter를 누릅니다.
7. 검색 결과를 누르면 해당 대화방으로 이동하고, 오른쪽 전체 대화에서 해당 메시지 위치가 강조됩니다.

채팅방 목록, 검색 결과, 대화 타임라인은 필요할 때 접고 펼칠 수 있습니다.
검색 결과와 대화 타임라인은 각 패널 제목 오른쪽의 원형 버튼으로 접습니다.
통합 뷰어는 사용자가 선택한 폴더 안의 `.json` 파일과 `_images` 폴더를 읽습니다.
백업 불러오기 버튼은 채팅방 목록 아래에 있습니다.
Chrome 또는 Edge에서 **백업 폴더 선택**을 사용하면, 브라우저 권한이 유지되는 동안 다음 실행 시 같은 폴더를 자동으로 다시 읽습니다.
발신자 이름이 없는 메시지는 내가 보낸 메시지로 보고 오른쪽에 표시하고,
발신자 이름이 있는 메시지는 다른 사람이 보낸 메시지로 보고 왼쪽에 표시합니다.
브라우저 보안 때문에 사용자가 선택하지 않은 폴더는 자동으로 읽을 수 없습니다.

더 자세한 화면 설명은 [통합 백업 뷰어 사용법](backup-viewer.md)을 참고하세요.

## 자주 막히는 문제

| 증상 | 해결 방법 |
| --- | --- |
| 어디서 다운로드해야 할지 모르겠음 | [ZIP 다운로드 링크](https://github.com/kangjung/kakaowork-backup/archive/refs/heads/main.zip)를 직접 누르세요 |
| `Run-Exporter.cmd`가 보이지 않음 | ZIP 파일 안이 아니라 압축을 푼 폴더를 열었는지 확인하세요 |
| 결과에 오래된 대화가 부족함 | **최대 스크롤 횟수**를 더 크게 해서 다시 실행하세요 |
| 실행 시간이 너무 오래 걸림 | **중지** 버튼을 누르면 지금까지 수집한 내용으로 저장됩니다 |
| 통합 뷰어에 이미지가 안 보임 | JSON 파일과 같은 이름의 `_images` 폴더까지 함께 선택했는지 확인하세요 |
| 카카오워크 창을 찾을 수 없다는 오류 | 카카오워크를 실행하고 잠금을 해제한 뒤 다시 시도하세요 |
| 대화방을 열라는 오류 | 백업할 대화방을 클릭해서 메시지가 보이는 상태로 둔 뒤 다시 실행하세요 |
| 회사 PC에서 실행이 막힘 | 회사 보안 정책 때문일 수 있습니다. 보안 담당자 또는 IT 담당자에게 확인하세요 |

## 보안 안내

업무 대화에는 회사 기밀, 개인정보, 고객 정보가 포함될 수 있습니다.
백업하기 전에 회사의 정보보안 정책과 데이터 반출 정책을 먼저 확인하세요.
